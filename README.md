# CLI приложение с разными функциями

## Установка
1. linux 🐧

Необходимо ввести одну команду
```
./install.md
```

## Функционал

### Погода
По умолчанию выводится Москва
```bash
ctl weather
```
```bash
Город: Москва
Температура: 19.1°C
Состояние: Местами дождь
```
Можно также выбрать любой город
```bash
ctl weather London
```

```bash
Город: London
Температура: 11.0°C
Состояние: Пасмурно
```


### Приветствие
```bash
ctl hello George
```

```bash
Hello, George!
```
